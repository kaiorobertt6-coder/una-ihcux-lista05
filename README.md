# una-ihcux-lista05
Neste atividade, foram aplicadas três principais heurísticas de usabilidade. A visibilidade do status é garantida por mensagens que informam ao usuário o que o sistema está fazendo, como “Conectando ao Banco Central” e “Calculando taxas”, evitando a sensação de travamento. A prevenção de erros é feita com o uso de try-catch, que impede o programa de encerrar inesperadamente e exibe uma mensagem clara quando o usuário digita valores inválidos. Já a estética e o design minimalista são trabalhados com o uso de cores e uma formatação organizada no console, destacando o resultado final e exibindo o valor com duas casas decimais, tornando a leitura mais clara e profissional.
